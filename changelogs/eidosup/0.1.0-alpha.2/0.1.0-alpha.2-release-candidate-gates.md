# Eidosup release candidate gates

- Added deterministic checksum and release metadata generation for Eidosc and
  Eidosup release assets.
- Added complete RID-set, digest, size, and safe ZIP structure verification
  before publication.
- Added isolated Windows x64 and Linux x64 Eidosup installation tests that run
  `eidosc info` and compile a tutorial smoke source from the installed bundle.
- Changed release workflows so tags and public prereleases are created only
  after candidate verification succeeds.
