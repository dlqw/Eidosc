# Eidosup 0.1.0-alpha.2 distribution bootstrap

- Added a public GitHub release-source boundary with environment-only authentication for private custom sources.
- Added strict SemVer 2.0.0 and deterministic `stable`/`preview` channel resolution.
- Added stable structured errors, redacted verbose diagnostics, machine-readable error output, and documented exit codes.
- Added doctor check IDs, severity, remediation, JSON output, and an unhealthy exit contract.
