# Eidosup 0.1.0-alpha.2 verified atomic toolchain installation

- Require `SHA256SUMS` coverage for the selected host bundle and verify its
  declared size and SHA-256 digest before installation.
- Add resumable bounded downloads, finite transient retries, and a revalidated
  content-addressed artifact cache.
- Reject unsafe ZIP paths, links, conflicting entries, special files, and
  excessive archive expansion.
- Record and rehash the complete installed file set in an installation manifest.
- Serialize setup operations and use staging, atomic directory replacement,
  rollback copies, and durable transaction journals for interruption recovery.
- Keep setup dry runs free of install, download, cache, lock, and transaction
  filesystem changes.
