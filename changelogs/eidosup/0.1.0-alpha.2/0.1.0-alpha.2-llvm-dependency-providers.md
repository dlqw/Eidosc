# Eidosup 0.1.0-alpha.2 LLVM dependency providers

- Define the supported LLVM/Clang 18-22 range and required `clang` and
  `llvm-ar` commands as dependency metadata.
- Separate dependency probing, provider planning, execution, and post-install
  verification.
- Add deterministic non-interactive plans for Windows, Linux, and macOS package
  managers, including explicit elevation handling.
- Refuse to replace an installed incompatible LLVM version implicitly.
- Report combined LLVM compatibility through `eidosup doctor` and stable
  dependency error codes.
