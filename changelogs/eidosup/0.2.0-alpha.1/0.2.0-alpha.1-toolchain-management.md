# Eidosup 0.2.0-alpha.1 toolchain management

- Add verified `toolchain install`, `list`, and transactional `uninstall`
  commands for channel and exact-version selectors.
- Add explicit global `default` selection and clearing without changing PATH.
- Add `update` and non-mutating `check` commands for all installed channels or
  requested toolchain specifications.
- Add structured `show`, `show active-toolchain`, `show home`, and fixed
  bootstrap-profile reporting.
- Add explicit `run` and `which` commands using the same verified resolver and
  process-forwarding contract as the stable shim.
- Record channel activation history and add rollback to retained verified
  immutable installations.
- Serialize lifecycle commits with a per-root management lock and recover
  interrupted uninstall transactions before subsequent mutations.
- Add machine-readable output, dry-run planning, CLI contract coverage,
  concurrent two-version installation tests, and uninstall fault-injection
  recovery tests.
- Resolve management roots from `EIDOS_HOME` unless an explicit
  `--install-root` override is supplied.
