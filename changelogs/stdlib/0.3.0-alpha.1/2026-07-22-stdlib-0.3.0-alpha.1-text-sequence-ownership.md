# Eidos Std 0.3.0-alpha.1 — text and sequence ownership

- Borrow text and sequence values for internal length, indexing, comparison, search, and traversal helpers that read the same owned value more than once.
- Make public `Seq.len` and `Seq.is_empty` shared-reference observers, and migrate container metadata observers and their callers to explicit `Ref` arguments.
- Add reference-based intrinsics for string equality and sequence length so standard-library implementations no longer depend on body-derived Copy behavior.
