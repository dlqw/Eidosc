# Eidosc 0.5.0-alpha.1 compiler metaprogramming

- Added the compiler-built-in `Type` value and versioned `Meta` schema for read-only semantic type/declaration reflection and explicit-target layout reflection.
- Added typed `Meta::Expansion` declaration, expression, pattern, handle, diagnostic, and module-member builders without source-string insertion or mutable AST exposure.
- Added user-defined `@derive` generators with imported generator resolution, fixed-point execution, cycle/identity diagnostics, hygiene handles, stable generated origin, and cache serialization.
- Added deterministic comptime fuel, canonical allocated-value-byte, and diagnostic budgets plus call/query/builder/diagnostic/cache tracing.
- Added generated symbols to ordinary resolution, type checking, trait coherence, HIR/MIR lowering, completion, hover, definition, references, and stable virtual documents.
- Added `eidosc meta expand` with readable/JSON output, generated document emission, comptime tracing, and budget overrides, plus the `eidos/generatedDocument` LSP request.
