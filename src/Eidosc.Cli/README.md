# Eidosc CLI

`Eidosc.Cli` is the command-line compiler and tooling entrypoint for Eidos.

## Common commands

```powershell
eidosc info --version
eidosc build <file-or-project>
eidosc run <file-or-project>
eidosc fmt <file>
```

For one-shot environment bootstrap on end-user machines, use the separately published `eidosup` bootstrap tool.
